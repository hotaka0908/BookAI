# AIデバイス未来考察

「喋ったらなんかいい感じにやってくれる」——そんな未来がもう始まっている。

本書は、AIグラス・AIネックレス・AIイヤホンなど急速に登場するAIデバイスを網羅的に紹介し、その設計思想（入力・推論・記憶・出力）を解説。MetaやApple、OpenAIが狙う次世代デバイスの姿を考察する。

アプリが消え、仕事が変わり、教育が変革する——AIデバイスがもたらす社会変化とは？ 現在AIデバイスを開発中の著者が、技術と哲学の両面から「iPhoneの次」を見通す未来考察。
